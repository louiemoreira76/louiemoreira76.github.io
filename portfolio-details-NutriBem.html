Sobre o projeto

O NutriBem é o projeto integrador do 3° semestre, um sistema completo para gestão de clínicas de nutrição, desenvolvido como trabalho acadêmico. Ele oferece três perfis de acesso:

  • Paciente: agenda consultas, visualiza histórico, edita perfil.
  • Nutricionista: gerencia agenda, registra evoluções, acessa prontuários.
  • Recepcionista: cadastra pacientes e nutricionistas, gerencia horários.

O backend foi construído com Spring Boot, utilizando autenticação, validações robustas, tratamento global de exceções e banco em memória H2. O frontend em React + Vite consome a API via Axios e possui rotas protegidas, componentes reutilizáveis e notificações com React Toastify e SweetAlert2.

Este projeto demonstra minhas habilidades em desenvolvimento full‑stack, modelagem de banco de dados, segurança e criação de interfaces responsivas e intuitivas.
