Sobre o projeto

O GameSync foi meu Trabalho de Conclusão de Curso (TCC) do ensino técnico. A plataforma conecta jogadores a um catálogo dinâmico de jogos, permitindo explorar títulos por gênero, avaliar, adicionar ao carrinho e finalizar compras (simulação de pagamento).

  • Frontend React: interface imersiva com carrosséis, animações 3D (Three.js) e design responsivo.
  • Backend Node.js + Express: API RESTful para gestão de usuários, jogos e pedidos, com upload de imagens via Multer e banco de dados MySQL.
  • Área administrativa: cadastro e edição de produtos, controle de pedidos e notícias.

O projeto foi desenvolvido inicialmente em dois repositórios separados (TCC-REACT e TCC-DB) e posteriormente unificado no repositório GameSync com melhorias na organização do código, autenticação e experiência do usuário.
