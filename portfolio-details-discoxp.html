Sobre o projeto

A DiscoXP é o projeto integrador do 4° semestre, é uma plataforma completa de e-commerce focada na venda de jogos digitais e físicos. O sistema oferece:

  • Catálogo de produtos com imagens, avaliações e efeitos 3D
  • Carrinho de compras persistente (IndexedDB)
  • Área do cliente: perfil, endereços, histórico de pedidos
  • Painel backoffice com perfis diferenciados (admin/estoquista)
  • Autenticação e validações robustas (CPF, e-mail, etc.)

O backend foi desenvolvido em Java Spring Boot, seguindo os princípios REST, com DTOs, tratamento global de exceções, testes de integração e segurança. O frontend utiliza React + Vite, com componentes estilizados via Styled Components, roteamento, consumo de API via Axios e efeitos interativos (Three.js, Framer Motion).
