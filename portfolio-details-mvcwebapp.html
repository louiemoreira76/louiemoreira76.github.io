Sobre o projeto

Aplicação web feita para passar no processo seletivo do estágio, desenvolvida com ASP.NET Core MVC e Entity Framework para gestão de produtos e categorias. O sistema permite cadastro, edição, exclusão e listagem, com validações e interface responsiva usando Bootstrap. Banco de dados SQL Server com migrations.

Esse projeto foi um dos primeiros contatos com o padrão MVC em C#, demonstrando habilidades com ORM, rotas, views e controle de sessão. Acesse o repositório no GitHub para mais detalhes.
